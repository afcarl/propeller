{% extends "layout.html" %} {% block header %}RESTful API{% endblock %} {% block content %} You can classify pictures by using example code from our GitHub page: 
  python classify.py -s research.telenor.io -u USER INPUT
where INPUT is either a URL to an image, or the path to a local text file with URLs. If you input a URL, the client will return the classification immediately, otherwise the return string will be how many pictures are queued for classification. USER is the user for the image. In order to retrieve the classifications, you can retrieve this with curl for instance: 
curl -u telenor:research research.telenor.io/images/prediction/USER/URL
If you want to retrieve all the pictures of dogs for a given user, you can do it like this: 
curl -u telenor:research research.telenor.io/images/prediction/USER/category/dog
