{% extends "layout.html" %} {% block header %}RESTful API{% endblock %} {% block content %} You can classify pictures by using example code from our GitHub page. First, you must ask the server to classify an image. This you can do the following way: 
python classify.py -g <GROUP> <INPUT>
where <INPUT> is either a URL to an image, or the path to a local text file with URLs. If you input a URL, the client will return the classification immediately, otherwise the return string will be how many pictures are queued for classification. <GROUP> is how you define your users/groups. For instance, for our web server, we use the group web, but you can use your own name, the city name, etc.

After you asked the server to classify an image, you can retrieve the classification with curl like this: 
curl -u <USERNAME>:<PASSWORD> research.telenor.io/images/archive/<GROUP>/<URL>
Where <USERNAME>:<PASSWORD> are the same you used to log onto this server. If you want to retrieve all the pictures of dogs for a given user, you can do it like this: 
curl -u <USERNAME>:<PASSWORD> research.telenor.io/images/archive/<GROUP>/category/dog
